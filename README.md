这个项目主要就是 拿到别人推送过来的项目数据，搭建后台管理系统
项目使用了SpringBoot作为项目基础框架，整合mybatis-plus，Sa-token，websocket，aliyunSDK
完成了权限划分，消息通信，短信服务，用户管理
